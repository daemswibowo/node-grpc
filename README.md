# GRPC Implementation Example
This project for simple example usecase with GRPC with nodejs.

# Project Structures
```
grpc
├── service-1-server
└── service-2-client

2 directories
```
# Start gRPC Server on service 1
```bash
npm run grpc:start
```
# Start expressjs on service 2 (consuming service-1-server customers)
```bash
npm run dev
```
